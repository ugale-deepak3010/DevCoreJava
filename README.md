# SOLID Principles.

## Single Responsible 

Employee class should not have salary related logics. Student class should not have exam related logic.

## Open Closed 

Open for extension, Closed for modification.

## Liskov 

A Penguin should not extend Bird if Bird guarantees flying behavior.

## Interface Segragation

A Printer interface should not force Scanner functionality on simple printers.
